a. ¿Quién inventó el sistema de control de versión Git y por qué?
 -El sistema de control de versión Git fue creado por Linus Torvalds para lograr la compatibilidad del mantenimiento de versiones de aplicaciones cuando tienen un gran número de archivos de código fuente.
