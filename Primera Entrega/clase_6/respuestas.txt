a. ¿Quién inventó el sistema de control de versión Git y por qué?

Fue creado por  Linus Torvalds, con el objetivo de brindar eficiencia, confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones cuando estas tienen un gran número de archivos de código fuente

b. ¿A quién pertenece actualmente Github y por qué?

Actualmente pertenece a la multinacional tecnológica Microsoft desde el 4 de junio del 2018, siendo esta adquisición una de las más grandes estrategias para dar frente a las grandes competencia que presenta esta empresa dentro del mundo de la tecnológica.

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?

Por medio de IDE o editores de códigos fuentes como lo pueden ser Visual Studio, NetBeans, sublime Text, atom, entre otros, los cuales se caracterizan por brindar una interfaz grafica de lo se esta trabajando.
