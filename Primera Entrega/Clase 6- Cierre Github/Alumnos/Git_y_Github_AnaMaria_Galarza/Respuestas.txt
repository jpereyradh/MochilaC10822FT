a. ¿Quién inventó el sistema de control de versión Git y por qué?

El desarrollador se llama Junio Hamano, aparece como autor Linus Torvalds. Git surgio porque: Linux tenía un gestor de paquetes que pasó a ser de pago, por lo que el desarrollador principal de Linux decidió crear un nuevo gestor de control de versiones para su sistema, y que diera soporte a su core. De esta forma nació Git

b. ¿A quién pertenece actualmente Github y por qué?

El 4 de junio de 2018, Microsoft compró GitHub por la cantidad de 7500 millones de dólares.

c. ¿Hay otra forma que no sea la terminal para trabajar con Github?

Si, hay programas con interfas grafica como: Gitkraken o GitHub Desktop
