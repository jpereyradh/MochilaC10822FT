Comandos para trabajar con Git y GitHub

cd: change directory
	cd ../ : cambia de directorio hacia una carpeta anterior
	cd ./ : cambia el directorio hacia una carpeta dentro de nuestro directorio

pwd: print working directory (muestra en consola el directorio en el cual estamos parados actualmente)

ls: lista los archivos y directorios que tengamos dentro del directorio actual 
	ls -R: Muestra todo la ruta del directorio revelando cada directorio con sus archivos dentro de la misma.
	ls -a: Muestra archivos ocultos.

touch: crea archivos
nano : edita archivos
cp: copia arhivos
mv: mueve archivos si existen ambas direcciones, de lo contrario la renombra

mkdir: crea un directorio
rmdir: remueve/elimina un directorio

rm: remueve/elimina un arhivo








----Git y GitHub----

git add . : agrega todos los cambios al staging area.
git commit -m "mensaje" :se confirman los cambios con un mensaje de referencia.

git status: permite ver el estado de los archivos con respecto al directorio.

git pull: actualiza el repositorio local, trayendo todos los archivos que fueron modificados en el repo remoto (en el origin)
	git pull origin main : te trae los archivos del repo remoto de la rama 'main'.
	git pull origin ramiro: te trae los archivos del repo remoto de la rama 'ramiro'.
