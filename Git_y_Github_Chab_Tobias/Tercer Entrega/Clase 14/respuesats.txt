Framework's
Luminus es un micro framework de clojure que esta basado en librerias livianas, el mismo provee robustes, esacbilidad,
y un uso intituivo, en consecuencia, podremos programar sin ningun problema. Se pueden hacer api restful, templates,
websocket, middleware, es decir, es muy completo para backend.

Javalin es un framework liviano de kotlin el cual soporta websockets, HTTP2, y peticiones asincronas.
